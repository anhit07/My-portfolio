Hotel Management System

  • Full Stack Web application with MEAN stack

Hotel Management System is a website used to manage the hotel information with Restful services including the following functionalities: User registration and authentication, displaying the list of the existing hotels, displaying, creating and updating information for a particular hotel. and displaying, creating and updating the customer reviews for a hotel.

Built with:

Front-end: HTML5, CSS3, AngularJS | Back-end: NodeJS | Database: NoSQL - MongoDB
